# TeX edit environment on DevContainer

DevContainerを使用したTeX編集環境のテンプレートです。

## 使い方

### 1. リポジトリをフォークする

GitHubでこのリポジトリをフォークしてください。

### 2. ローカルにクローンする

フォークしたリポジトリを自分のローカル環境にクローンします。

```bash
git clone https://github.com/your-username/tex-template.git
cd tex-template
```

### 3. VS Codeでワークスペースを開く

VS Codeでクローンしたリポジトリのフォルダをワークスペースとして開いてください。

### 4. DevContainerで再度開く

VS Codeの右下に「DevContainerとして再度開く」という案内が表示されるので、これを許可してください。これにより、TeX編集に必要な環境が自動的にセットアップされます。
